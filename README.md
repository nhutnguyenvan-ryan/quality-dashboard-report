# QA Dashboard Automation Platform
Hệ thống hiển thị báo cáo trực tuyến cho QA Monitoring (Shopee E-commerce).

## Hướng dẫn cấu hình trên Render:
- **Build Command**: `cd backend && npm install`
- **Start Command**: `cd backend && npm start`
- **Environment Variables (Advanced)**:
  - `SPREADSHEET_ID`: ID của Google Sheet
  - `GSHEETS_CLIENT_EMAIL`: Email Service Account
  - `GSHEETS_PRIVATE_KEY`: Private Key bảo mật
